# Task 3: Deep-Dive Analysis & Interactive Dashboard

## 📌 Objective

To solve complex business problems through deep-dive analysis and build
an interactive dashboard that highlights key insights and supports
decision-making.

------------------------------------------------------------------------

## 📊 Project Overview

This project focuses on analyzing revenue performance, customer
behavior, and business trends using data analytics and visualization
tools.

------------------------------------------------------------------------

## 🔑 Key KPIs

-   **Total Revenue:** £2M\
-   **Total Customers:** 332\
-   **Churn Cases:** 162\
-   **Average Order Value (AOV)**\
-   **Customer Segmentation (High / Medium / Low)**

------------------------------------------------------------------------

## 📈 Analysis Performed

### 1. Revenue Trend Analysis

-   Revenue grew steadily from 2023 to 2025\
-   Decline observed in 2026\
-   Indicates possible market or demand changes

### 2. Product Category Analysis

-   **Top Performers:** Sports, Clothing\
-   **Underperforming:** Electronics\
-   Opportunity to improve weaker categories

### 3. Customer Segmentation

-   High-value customers generate most revenue\
-   Medium segment has highest volume\
-   Conversion opportunity from Medium → High

### 4. Payment Method Insights

-   Cash, Card, and UPI almost equally used\
-   Shift towards digital payments

### 5. Age Group Analysis

-   Majority customers: 25--45 years\
-   Core target audience

### 6. Quarterly Trends

-   Q2 shows dip\
-   Q3 and Q4 recovery\
-   Indicates seasonal trends

------------------------------------------------------------------------

## 🚀 Key Insights

-   Strong revenue but high churn risk\
-   High-value customers drive growth\
-   Electronics category needs improvement\
-   Medium customers are conversion opportunity

------------------------------------------------------------------------

## 🎯 Recommendations

-   Implement churn reduction strategies\
-   Focus on high-value customer retention\
-   Improve underperforming product categories\
-   Optimize marketing for 25--45 age group
